This project was completed in Spring 2026 for my Discovery & Learning of Big Data class. It focuses on predicting hospital readmission risk using healthcare and patient data analytics techniques. The goal is to identify patients who may be at higher risk of being readmitted after discharge, allowing healthcare providers to improve care planning, reduce costs, and enhance patient outcomes. Using machine learning models and data preprocessing techniques, the project analyzes factors such as patient demographics, diagnoses, length of stay, medical history, and clinical indicators to generate predictive insights. The workflow includes data cleaning, exploratory data analysis (EDA), feature engineering, model training, evaluation, and visualization of results.

Key Features :
  Data cleaning and preprocessing
  Exploratory data analysis (EDA)
  Predictive modeling using machine learning algorithms
  Model performance evaluation and comparison
  Data visualization and insight generation
  
Technologies Used :
  Python
  Pandas & NumPy
  Scikit-learn
  Matplotlib & Seaborn
  Jupyter Notebook
  
Objective :
The purpose of this project is to demonstrate how data analytics and machine learning can support proactive healthcare decision-making by identifying patients at risk for hospital readmission and improving overall healthcare efficiency.
